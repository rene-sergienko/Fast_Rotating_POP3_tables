# Integrated Stellar Yield Tables

This GitHub Pages site hosts interactive, searchable, sortable tables of integrated stellar yields.

Files:
- `index.html`: web page containing all four interactive tables
- `yields_lowmass.csv`: total yields, 5–50 solar masses
- `yields_highmass.csv`: total yields, 60–200 solar masses
- `yields_lowmass_windonly_30_50.csv`: wind-only variant for 30–50 solar masses
- `yields_highmass_windonly_60_90.csv`: wind-only variant for 60–90 solar masses
